# D.support ウェブサイト構成案

## サイト概要
- **サービス名**: D.support（ディーサポート）
- **タグライン**: 地域医療のために
- **事業内容**: クリニックの開業・閉院をトータルサポート
- **テーマカラー**: ブルー系（ロゴに合わせたグラデーション）
  - メインカラー: #1e4a7a（濃い青）
  - アクセントカラー: #3d7ab8（明るい青）
  - サブカラー: #e8f4fc（淡い青）

---

## ページ構成

### 1. トップページ (index.html)
**ヒーローセクション**
- キャッチコピー: 「クリニックの開業から閉院まで、地域医療を支えるパートナー」
- サブコピー: 経験豊富な専門家が、先生の新しいスタートをサポートします
- CTAボタン: 「無料相談を予約する」「サービス内容を見る」

**サービス概要セクション**
- 開業サポート（カード形式）
- 閉院サポート（カード形式）
- 継承サポート（カード形式）

**選ばれる理由セクション**
1. 医療専門のコンサルタント在籍
2. 開業から閉院までワンストップ対応
3. 地域医療への貢献を第一に
4. 豊富な実績とネットワーク

**サポート実績セクション**
- 数字で見る実績（相談件数、開業支援数、閉院支援数など）

**お客様の声セクション**
- 実際に利用された医師の声（3〜4件）

**よくある質問セクション**
- 開業・閉院に関する基本的なFAQ（5〜6項目）

**CTAセクション**
- 無料相談への誘導

---

### 2. 開業サポートページ (/kaigyou/)

**ヒーローセクション**
- 「理想のクリニック開業を実現」

**開業までの流れ**
1. 初回相談・ヒアリング
2. 事業計画の策定
3. 物件選定・契約
4. 内装・設備の手配
5. 行政手続き・届出
6. スタッフ採用
7. 開業準備・広報
8. 開業

**提供サービス詳細**
- 事業計画書作成支援
- 資金調達サポート（銀行融資、リース等）
- 物件探し・契約交渉
- 内装設計・医療機器選定
- 行政手続き代行
- 採用支援
- 集患マーケティング

**開業までのスケジュール目安**
- タイムライン形式で表示

**開業に必要な届出一覧**
- 保健所、厚生局、税務署等への届出リスト

---

### 3. 閉院サポートページ (/heiin/)

**ヒーローセクション**
- 「閉院手続きを、負担なくスムーズに」

**閉院を検討される先生へ**
- 閉院の選択肢（完全閉院 vs 継承）の比較
- 費用の目安（数百万円〜1,000万円程度）

**閉院までの流れ（タイムライン）**
| 時期 | やるべきこと |
|------|-------------|
| 6ヶ月前 | 閉院決定、専門家相談、計画策定 |
| 3ヶ月前 | スタッフへの通知、患者への告知開始 |
| 2ヶ月前 | 行政手続き開始、医療機器処分手配 |
| 1ヶ月前 | 患者紹介状作成、最終届出 |
| 閉院日 | 最終診療、各届出提出 |
| 閉院後 | カルテ保管、残務処理 |

**提供サービス詳細**
- 閉院計画の策定
- 行政手続き代行（保健所、厚生局、税務署等）
- スタッフ退職手続きサポート
- 患者引継ぎ支援
- 医療機器・備品の売却・処分
- 不動産契約の解約・原状回復
- カルテ保管サポート

**必要な届出一覧**
- 届出先、届出書類、期限をテーブル形式で表示

**閉院にかかる費用の目安**
- 原状回復費用
- 医療機器処分費用
- 退職金・未払賃金
- 行政書士・税理士費用

---

### 4. 継承サポートページ (/keishou/)

**ヒーローセクション**
- 「地域医療を次世代へつなぐ」

**継承のメリット**
- 閉院コストがかからない
- 医院譲渡による対価が得られる可能性
- 患者・スタッフを守れる
- 地域医療の継続に貢献

**継承の流れ**
1. 初回相談・価値評価
2. 後継者探し
3. 条件交渉
4. 契約締結
5. 引継ぎ期間
6. 継承完了

**継承 vs 閉院 比較表**
- 費用、期間、患者への影響などを比較

## 技術仕様（推奨）

- **フレームワーク**: Next.js または Astro（静的サイト向け）
- **スタイリング**: Tailwind CSS
- **デプロイ**: Vercel または Netlify
- **フォーム**: Formspree または自社バックエンド
- **レスポンシブ**: モバイルファースト設計
